# 实验室记账软件开发

## 定义

针对实验室记账的需求，开发记录软件

### 需求设计

- 可以存储账单，并对账单进行类型定义
- 支持账单 tag 功能
- 支持账单用户设计
- 支持存储对应发票文件
- 支持账单时间记录
- 支持账单关键字查找
	- 支持账单互相引用
- 支持账目导出功能
- 具备友好的用户 UI

###### 账单

- ID
- 时间
- 简介
	- 一句话简述
- [人员](./)
	- **一个**
		- 只记录主要人员
- 商品类别 (Tag 功能)
	- Enum
		- Food
		- Entertainment
		- Equipment
		- Knowledge
		- Wage
		- Other
- 价格
	- decimal
- 详细信息
- 订单类型
	- 可报销
	- 不可报销
- 订单状态
	- 完成
		- 注意，不可报销默认完成
	- 未完成
		- 实际上就是未报销

	- 订单截图
