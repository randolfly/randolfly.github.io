::: tip definition
定义$P_m\left( F \right)$
表示对于非负整数m，$P_m\left( F \right)$表示系数在F中且次数不超过m的所有多项式构成的集合
:::


::: note note
记号**F、V、W**
- F表示R或者C
- V和W表示F上的向量空间
:::
